Firefox is Open Source

Why does it matter?

Proprietary

Built internally by a monolithic company.

Single contributer, closed development.

Seeks users, not community involvment.

Code kept hidden, possibility of hidden backdoors.

Opensource

Built by an open community of users.

Many contributors, open development

Encourage user involvement

Code available for evaluation, no hidden backdoors.
